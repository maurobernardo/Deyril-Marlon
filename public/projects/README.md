## Adding New Projects

When adding a new project:

1. Create a folder with the project ID: `public/projects/[project-id]/`
2. Add images named `image0.jpg`, `image1.jpg`, etc.
3. Optionally add a `video.mp4` file
4. Update the projects array in `components/Projects.tsx`
5. Add the project detail page in `app/projects/[id]/page.tsx`
